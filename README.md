# Space Explorer - Animated Landing Page

## Overview

Space Explorer is a responsive landing page developed using HTML5 and CSS3. The project demonstrates the use of CSS animations, transitions, and modern web design techniques to create an engaging and interactive user experience.

This project was created as part of Task 8: Add Animations to a Webpage.

## Features

- Responsive Design
- Animated Navigation Header
- Floating Rocket Animation
- Rotating Earth Animation
- Fade-In Hero Section
- Slide-Up Feature Cards
- Image Hover Zoom Effects
- Smooth CSS Transitions
- Mobile-Friendly Layout

## Technologies Used

- HTML5
- CSS3
- CSS Animations
- CSS Transitions
- Flexbox
- Media Queries

## Animations Implemented

### Slide Down Animation
Used for the header section when the page loads.

### Fade In Animation
Applied to the hero section content.

### Floating Rocket Animation
The rocket continuously moves up and down using CSS keyframes.

### Slide Up Animation
Feature cards smoothly appear from the bottom.

### Rotating Earth Animation
The Earth image rotates continuously using CSS animation.

### Hover Effects
Images, cards, and buttons enlarge smoothly when hovered over.
